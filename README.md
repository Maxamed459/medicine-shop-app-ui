# Medicine App

Medicine App is a Flutter mobile application for managing and tracking medications.  
Use it to keep an organized list of your medicines, dosages, schedules, and other related information.

## Features

- **Medication list**: View and manage a list of medicines.
- **Schedules & reminders**: Configure when medications should be taken.
- **Details view**: Store notes such as dosage, frequency, and instructions.
- **Modern UI**: Clean and user-friendly interface built with Flutter.

> Note: The actual feature set depends on the current implementation of the project and can be extended.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- **Flutter SDK** installed (`stable` channel recommended)
- **Dart** (bundled with Flutter)
- An IDE such as **Android Studio**, **VS Code**, or **IntelliJ IDEA**

### Installation

1. **Clone the repository**
   ```bash
   git clone <this-repo-url>
   cd medicine_app
   ```

2. **Get dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
